# Simple Registry

## Description

A contract that that get information from registry

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/)
- [MetaMask](https://metamask.io/) browser extension installed

### Installation

1. Clone the repository.
2. cd simple-registry-frontend
3. yarn install
4. yarn dev
